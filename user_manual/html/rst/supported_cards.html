Supported Nastran cards¶

This section provides an overview of Nastran cards and their fields that are supported as experiment parameters / features in ACE.

To select a Nastran card and field as an experiment parameter in the ACE project file (*.ace) simply use the card method, e.g. selection of field T of two card PSHELLs is done as follows:

# Define field T of PSHELL cards with ID 10 and 45 as an feature
# Note that the variable name "pshell_features_example" is arbitrary
pshell_features_example = card(name='PSHELL', ids=[10, 45], field='T')

# Define field V1 of PARAM card with ID AMLSNCPU as an feature
# Note that the variable name "param_amlsncpu_example" is arbitrary
param_amlsncpu_example = card(name='PARAM', ids='AMLSNCPU', field='V1')
