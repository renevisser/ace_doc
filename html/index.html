CDH/ACE Documentation¶

The CDH/ACE documentation is available as html based webpages (recommended) as well as static pdf files for offline reference. For efficient nagivation all documentations contain internal cross-references to sections, figures and tables. Furthermore a navigation menu is available on the left hand side of each manual. For html based document versions also a quick search option is available to look for specified keywords or items. To return to the beginning of an html manual the logo in the upper left corner can be pressed.

Available manuals¶

The CDH/ACE documentation consist of the following manuals each providing information for a distinct purpose. The installation manual is intended for installation and system-wide configuration of the CDH/ACE software. The user manual describes in detail the way to use ACE and the options that are available whereas the example manual provides easy step-by-step demonstrations on ACE usage by means of simple example problems.

Links to specific CDH/ACE manuals available either as html website or pdf download:

  • User reference guide: user_manual (pdf download: user_manual.pdf)

  • Examples and tutorials: example_manual (pdf download: example_manual.pdf)

  • Installation manual: installation_manual (pdf download: installation_manual.pdf)
