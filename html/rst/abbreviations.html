Abbreviations¶

Following abbreviations are used frequently throughout the ACE documentation as well as in some of the log and result files. In this list all abbreviations are in CAPITALS however in the actual manuals they are often used in lowercase as well.

Common Abbreviations¶

Abbreviation

Meaning

Brief Description

ACE

Autonomous Computational Experiments

Name of this software package

CAE

Computer Aided Engineering

FEM

Finite Element Method

The most common numerical approach to 3D mechanical analysis

NVH

Noise Vibration Harshness

The science concerning the field of vibration and acoustics

AI

Artificial Intelligence

ML

Machine Learning

DOE

Design of Experiments

FRF

Frequency Response Function

A frequency spectrum describing the linear dynamic behavior of a system

ROM

Reduced Order Model

ML based surrogate model e.g. for fast response predictions

MSE

Mean Square Error

RAE

Relative Absolute Error

RMS

Root Mean Square

LHS

Latin Hypercube Sampling

SV

Simulation Variant

Each (Nastran) CAE job is referred to as a simulation variant

LinREG

Linear Regressor

A standard linear regressor algorithm

SVR

Support Vector Regressor

GPR

Gaussian Process Regressor

RFR

Random Forest Regressor

XGBR

eXtreme Gradient Boosting Regressor

An fast implementation of a gradient boosting algorithm intended for large-scale problems

LGBR

Light Gradient Boosting Regressor

A light-weight implementation of a gradient boosting algorithm

MLPR

Multi-Level Perceptron Regressor

A simple feed-forward neural network

CSV

Comma Separated Values

A common file format used to store text data in tables

PNG

Portable Network Graphics

A common picture format for saving high quality images
